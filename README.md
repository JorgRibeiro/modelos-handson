# Meus Modelos de ML baseados no Livro Hands-on

## Sobre este repositório

Este repositório contém implementações práticas de modelos de Machine Learning baseados no livro **"Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow"** de Aurélien Géron. Aqui você encontrará scripts Python (`.py`) e notebooks (`.ipynb`) com exemplos, exercícios e experimentos relacionados aos capítulos do livro.

## O Agente de Aprendizado

Este projeto conta com um **agente de orientação personalizado** (`.agent.md`) projetado especificamente para auxiliar sua jornada de aprendizado em Machine Learning. 

### O que o agente faz:

- **Orienta por perguntas**: Em vez de fornecer respostas prontas, o agente faz perguntas guiadas para estimular seu raciocínio e compreensão.
- **Abordagem hint-first**: Começa com pistas e explicações conceituais; oferece pseudocódigo quando necessário; entrega código completo apenas quando solicitado explicitamente.
- **Explica conceitos**: Fornece explicações claras sobre por que um problema existe, como modelá-lo, quais soluções existem e seus trade-offs.
- **Trabalha com múltiplos formatos**: Suporta tanto notebooks (`.ipynb` / Colab) quanto scripts Python (`.py`), podendo analisar, corrigir e expandir ambos.
- **Foca na implementação**: Prioriza a prática com explicações conceituais, usando matemática apenas quando necessária para entender o conceito.

### Objetivo principal:

Ser um **parceiro de aprendizado** que não apenas resolve exercícios, mas **te ensina a pensar** como um cientista de dados—desenvolvendo sua capacidade de análise, implementação e resolução de problemas de forma independente e robusta.

### Como usar o agente:

Ao fazer uma pergunta relacionada a Machine Learning, exercícios do livro Hands-On ou projetos neste repositório, o agente pode:

- **Pedir mais contexto**: "Qual foi seu primeiro pensamento?" ou "O que você tentou até agora?"
- **Fazer perguntas orientadas**: "Que tipos de algoritmos você poderia usar para este problema?"
- **Fornecer pistas**: "Dica: considere usar validação cruzada para avaliar seu modelo."
- **Entregar código** (quando solicitado): Se pedir explicitamente "mostre o código" ou "implemente", receberá a solução pronta.

## Como começar

1. Explore os scripts e notebooks neste repositório.
2. Trabalhe nos exercícios do livro Hands-On.
3. Quando tiver dúvidas, faça perguntas ao agente — ele orientará seu aprendizado.
4. Se preferir uma solução pronta, peça explicitamente; caso contrário, receberá pistas e orientações.
